# 🏥 Emergency Tycoon

**Emergency Tycoon** es un simulador de gestión hospitalaria donde tomas el rol de administrador de una sala de emergencias. Deberás atender pacientes, gestionar recursos limitados y tomar decisiones que pueden salvar (o no) vidas.

## 🎮 Características

- Sistema de diagnóstico dinámico con prioridades.
- Doctores con comportamiento variable 
- Pacientes con historial médico almacenado con tablas hash.
- Gestión de recursos (dinero, médicos, insumos, equipos).
- Condiciones de victoria y derrota.
- Interfaz gráfica utilizando **SFML**.
